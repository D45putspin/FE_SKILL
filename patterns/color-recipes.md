# Color Recipes

## 1) Premium Dark
- Background: near-black with subtle tint
- Surface: translucent elevated layers
- Accent: one vivid brand color
- Text: high contrast neutrals

Use when: fashion, media, creator products.

## 2) Clean Editorial Light
- Background: warm off-white
- Surface: white + soft border
- Accent: restrained, used sparingly
- Text: strong neutral greys

Use when: content-heavy products and docs.

## 3) Technical Console
- Background: charcoal / blue-black
- Surface: high contrast panels
- Accent: cyan/lime for active states
- Status colors: success/warn/error clearly separated

Use when: dashboards, ops tools, data-heavy UI.

## Rules
- Keep accent usage under ~15% of visible area.
- Verify contrast for text and interactive controls.
- Define hover/active/focus shades up front.
- Never rely on color alone for status meaning.
