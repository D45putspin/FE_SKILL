# Layout Recipes

## App Shell
- Sticky top bar with context actions
- Main content container with max-width
- Optional bottom nav for mobile-first apps

## Feed Layout
- Single-column cards on mobile
- 2-column split on desktop where useful
- Keep card rhythm consistent

## Dashboard Layout
- Left navigation + right content pane
- Summary row first, detail panels second
- Use progressive disclosure for advanced controls

## Rules
- Use one spacing scale across all pages.
- Avoid nested scroll containers unless necessary.
- Preserve tap targets >= 40px on mobile.
- Align edges consistently across modules.
