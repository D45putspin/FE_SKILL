# Acceptance Checklist

## Visual
- [ ] Clear hierarchy in 3-second scan
- [ ] Signature differentiator present
- [ ] Consistent spacing + radius + shadow language

## Interaction
- [ ] All key components have full state coverage
- [ ] Motion is purposeful and consistent
- [ ] Reduced-motion respected

## Accessibility
- [ ] Keyboard navigation works
- [ ] Focus-visible present everywhere
- [ ] Contrast meets WCAG AA minimum

## Performance
- [ ] Large assets optimized
- [ ] Route/component lazy-loading used where needed
- [ ] No obvious animation jank on mid-range devices

## Product
- [ ] Primary user workflow is obvious
- [ ] Empty/loading/error states are handled
- [ ] Copy is product-specific (no placeholder slop)
